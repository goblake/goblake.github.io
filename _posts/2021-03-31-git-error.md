## 문제 발생
검색해보니 여러 이유가 있는데 나의 경우는 Mac OS  업데이트를 해서 그런 것 같다.
업데이트를 하며 Xcode Command Line Tools 관련하여 이슈가 발생한 것이다.


## 문제 해결

```yaml
xcode-select --install
```


터미널에서 위의 명령어를 입력하면 설치할 것이냐는 화면이 뜬다.
그러면 승인을 누르고 설치하면 오류가 더 이상 발생하지 않는다.
